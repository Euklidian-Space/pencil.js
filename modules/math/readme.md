# Math

Wrapper for some mathematical helpers.


## Installation

    npm install @pencil.js/math


## Examples

```js
import * as Math from "@pencil.js/math";

Math.truncate(-99.9); // -99

// or
import { constain } from "@pencil.js/math";
constain(99, 0, 10); // 10
```
